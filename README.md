# vue-timepicker

Time picker component for Vue

# Screenshot
![pic](/images/pic.png)

# Installation

```shell
$ npm install vue-timepicker
```

# Usage

```js
var Vue = require('vue')
var TimePicker = require('vue-timepicker')

// or ES2015
import TimePicker from 'vue-timepicker'

new Vue({
  el: '#app',

  components: {
    TimePicker: TimePicker
  }
})
```

```html
<div id="app">
  <time-picker></time-picker>
</div>
```

# Development

```shell
$ npm install

$ npm run dev
```
